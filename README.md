# EyeTracker
- Architected a real-time cursor control system using OpenCV and MediaPipe, leveraging the Face Landmarker deep learning model for sub-millisecond ocular landmark detection.

- Developed a custom coordinate-mapping engine using PyAutoGUI to translate facial gestures into precise screen-pixel interactions, optimizing for low-latency responsiveness.

- Implemented signal processing and smoothing algorithms to mitigate cursor jitter, ensuring high-fidelity human-in-the-loop control.

- Designed a modular backend to handle concurrent streams of landmark data, demonstrating the ability to manage complex, real-time data pipelines.
